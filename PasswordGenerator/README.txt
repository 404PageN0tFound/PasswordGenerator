Python is required for the password generator to work. If you don't have Python simply download it by opening the Install Python folder, then the .exe file based on your computer's processor. To check your computer's processor, go to Settings, then press System, navigate to About at the bottom of the scrolling menu at the side, now check System type. Now you know what bit processor to OS you have. If you have a different-bit operating system and a different-bit processor, download the one based on your operating system for the best performance.
